Els meus Projectes

Projecte futur 1: Web personal amb e-commerce

En aquest projecte, m'agradaria crear una pàgina web personal amb una secció d'e-commerce on vendre els meus dissenys i projectes. El faré utilitzant HTML, CSS, i JavaScript, i amb una base de dades per gestionar els productes.

Projecte futur 2: Aplicació de seguiment de tasques

Aquesta aplicació permetria als usuaris gestionar les seues tasques diàries. Es podria fer amb una aplicació web que integre una base de dades per a les tasques i recordar a l'usuari quan han d'acabar-les.

Projecte futur 3: Joc educatiu per a xiquets i xiquetes

El meu objectiu és crear un joc educatiu per a xiquets i xiquetes on aprenguen a programar de manera divertida. Utilitzaré HTML5, CSS i JavaScript per crear aquest joc interactiu i atractiu per als més xicotets.
